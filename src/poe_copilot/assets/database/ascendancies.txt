Ascendancy class | Description | Ascendancy classes are subclasses that grant an access to an Ascendancy skill tree specific for the character class chosen.
Ascendancy class | Overview | Choosing an Ascendancy class after completing the Labyrinth for the first time allows the player to access a new Ascendancy skill tree specific to the class chosen. Each character class comes with three associated Ascendancy classes (except for Scion , who only has two) for a total of 20 Ascendancy classes.
Ascendancy class | Overview | The first time the player completes a version of the Labyrinth, the player will receive two Ascendancy skill points to allocate in the ascendancy skill tree. The first three Labyrinths can be unlocked via playing through the Acts, resulting in six Ascendancy skill points. Two more skill points are unlocked when the Eternal Labyrinth, which requires an Offering to the Goddess to access, is completed, which means players can spend a total of eight Ascendancy skill points. [2]
Ascendancy class | Ascendancy skill tree | The Ascendancy skill tree is a small skill tree that a character has access to when the Labyrinth has been completed. Each Ascendancy class has its own tree. It functions similarly to the larger passive skill tree ; the player earns Ascendancy skill points, which can be spent on allocating nodes granting passive bonuses. The Ascendancy skill tree is shown on the regular passive skill tree in a tab near the character's starting location.
Ascendancy class | Layout | Ascendancy passive skills are significantly more powerful than regular passive skills and will heavily impact the way a character is played. A character can only spend a maximum of eight points on the tree; however, the last two points are not available until significantly later when end-game maps can be cleared. Aside from the Ascendant 's, the skill trees also follow a certain format. Most classes have 12 to 16 passive skills in each tree, and each potent "notable" skill is gated by a generic, lesser node, so it takes two points to reach them. Certain notables require the player to allocate another notable first, so they take up four points. A character will only be able to have one four-pointer skill until the Eternal Labyrinth is cleared.
Ascendancy class | Layout | The Scion's Ascendancy classes follow a different format: some notables on the Ascendant and the Reliquarian have two minor passive requisites, resulting in the class having up to three notables with all 8 Ascendancy passive points. In a typical setup, the Ascendant will have up to two class notables and one notable that grants them a second starting point, and the Reliquarian can have one armour notable, one weapon notable, and one jewelry notable. Due to the tree layout, the Scion won't have an Ascendancy notable until Cruel Labyrinth. and the Ascendant won't be able to gain a second starting point until Merciless Labyrinth.
Ascendancy class | Refunding | Refunding one Ascendancy skill point requires five regular refund points. When all points are refunded, use the Altar of Ascendancy in the Labyrinth treasure room in any difficulty to select a new Ascendancy class.
Ascendancy class | Bloodline Ascendancy classes | Bloodline Ascendancy classes are secondary Ascendancy classes that are unlocked by completing certain endgame bosses or encounters.
Ascendancy class | Bloodline Ascendancy classes | They are not specific to any character class and do not affect your character's usual Ascendancy class. Bloodline Ascendancy passives also require Ascendancy skill points to allocate, sharing these points between the regular and Bloodline Ascendancy class.
